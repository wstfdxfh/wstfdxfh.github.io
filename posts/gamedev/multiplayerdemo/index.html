MultiPlayerDemo

July 16, 2025 · 李睿杰
Table of Contents
  • 改进笔记

一款多人设计游戏的Demo，使用UnrealOnlineSubSystemSteam实现互联网双人联机

游戏包含开始界面，等待大厅和正式游戏关卡，在关卡开始之前，玩家可以飞行浏览地图

游戏开始后，玩家可以拾取地图上的武器，包括步枪和火箭筒，进行对抗

HUD上记录了击杀数和被击败的次数，通过人头数量决定最后的胜者。

相关内容参考 相关课程，UE5.5进行开发。相关开发笔记可以参见Unreal网络多人开发和Unreal版本控制等

下载链接

注意，游玩时，需要两台不同的电脑主机，各自登录不同的Steam账号；

点击开始界面的Host主持游戏作为Listen Server，另一台点击加入后作为Client

因为使用Steam的公用ID，在高峰期可能会出现连接缓慢或者无法加入的问题。

void AProjectile::BeginPlay()
{
    Super::BeginPlay();

    if(Tracer)
    {
       TracerComponent = UGameplayStatics::SpawnEmitterAttached(Tracer, CollisionBox, NAME_None, GetActorLocation(), GetActorRotation(),
                                                                EAttachLocation::KeepWorldPosition);
    }
    if(HasAuthority())
    {
       CollisionBox->OnComponentHit.AddDynamic(this, &AProjectile::OnHit);
       CollisionBox->IgnoreActorWhenMoving(Owner, true);
    }
}
